Project Overview

Project Connect is building the framework for viral, monetized open-source teams with equitable remuneration for all contributors. Our platform combines trading journal functionality with skill-based challenges and performance-based revenue sharing.

Key Features

  • Comprehensive trading journal platform
  • Skill-based challenge system
  • Performance-based revenue sharing
  • Vibrant open-source community
  • Educational resources and mentorship

Target Audience

  • Retail traders looking to improve skills
  • Trading educators and coaches
  • Aspiring fund managers
  • Open-source contributors
  • Community builders

Getting Started

Get up and running with Project Connect in just a few simple steps.

1. Create an Account

Sign up for a free account on our platform to access all basic features.

GET /api/v1/users/register
Content-Type: application/json

{
  "username": "your_username",
  "email": "your_email@example.com",
  "password": "your_secure_password"
}

2. Set Up Your Profile

Complete your profile with your trading experience and preferences.

3. Start Journaling

Begin logging your trades using our comprehensive journal system.

4. Join Challenges

Participate in skill-based challenges to prove your abilities and earn rewards.

API Endpoints

// Create a new trade
POST /api/v1/trades

// Get all trades for user
GET /api/v1/trades

// Get specific trade
GET /api/v1/trades/{id}

// Update trade
PUT /api/v1/trades/{id}

// Delete trade
DELETE /api/v1/trades/{id}

Challenges

Our challenge system allows you to prove your trading skills and earn rewards based on performance.

Challenge Types

Consistency Challenge

Maintain positive P&L for 30 consecutive days.

Duration: 30 days Prize: $500
Risk Management Challenge

Achieve 15% return with <2% max drawdown.

Duration: 60 days Prize: $1,000
Adaptability Challenge

Profit in all three market conditions.

Duration: 90 days Prize: $2,000
Strategy Mastery Challenge

Demonstrate expertise in specific strategies.

Duration: Variable Prize: $5,000

Scoring System

Challenges are scored based on multiple criteria:

  • Profitability (30%): Risk-adjusted returns
  • Consistency (25%): Streak of positive periods
  • Risk Management (25%): Drawdown control and position sizing
  • Adaptability (20%): Performance across market conditions

API Reference

Integrate with our platform using our comprehensive API.

Authentication

// Obtain authentication token
POST /api/v1/users/login
Content-Type: application/json

{
  "email": "your_email@example.com",
  "password": "your_password"
}

// Response
{
  "token": "your_jwt_token",
  "expires_in": 3600
}

Rate Limiting

All API endpoints are rate limited to 1000 requests per hour per user.

Error Handling

{
  "error": {
    "code": "VALIDATION_ERROR",
    "message": "Invalid input data",
    "details": [
      {
        "field": "email",
        "message": "Invalid email format"
      }
    ]
  }
}
